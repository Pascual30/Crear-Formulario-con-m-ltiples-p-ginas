# Crear-Formulario-con-multiples-paginas
Este proyecto implementa un formulario multipágina sin base de datos, utilizando estructuras de datos en memoria (JSON) para almacenar temporalmente la información. Cada página representa una sección del formulario y permite agregar múltiples registros dependiendo del tipo de datos.

## Características principales

**5 páginas organizadas por secciones:**
Datos Personales (un registro)
Familiares (múltiples registros)
Condiciones Pre-Existentes (múltiples registros)
Internamientos Realizados (múltiples registros)
Resumen de los Datos Registrados

Navegación entre páginas (Anterior / Siguiente)
Almacenamiento temporal en estructuras tipo JSON
Vista final donde se muestran todos los datos introducidos
Validaciones básicas para evitar registros incompletos
Código modular y fácil de mantener
